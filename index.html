Proving Grounds

Variables

  • use let when you want to be able to reassign variables later on
  • use const when you don't want to change variables once they've been set

  • a variable is undefined when it has been declared but not assigned anything
  • a variable is null when it intentionally has no value

Strings

  • you can assign strings using " " or ' '

  • however, if you have multiple variables you need to print out, it is much easier to use ` ` 
    • ex: const greeting = `Hello, ${name}!`;
    • it's like f-strings but without the f
    • you can also alter the variables within the { } like add or minus, etc
    • furthermore, you don't need to add \n for line breaks, just add the line breaks within the text

  • you can concatenate strings using + or .concat() 
    • ex: let result = str1.concat(' ', str2); will join string 1, a space and string 2 together

  • you can append strings using +=

  • you can access characters in a string just like python, through index-based square brackets
    • to access the last character, do: word[word.length - 1]
  • to add newlines within a string, add them within the string statement.
    • let poem = "poem start \n poem end \n"
  • to include special characters like " " within a string add \"...\" within a string, add them within the string statement.
    • to include ' ', you only need to add one \, right before the ' '. so like this: \' '
  • you can grab a strings length with .length

Debugging

  • by putting an output within console.log(...) you can test your variables and ensure your code is running right

Operators

  • typeof operator allows you to see the data type of a variable. it will print out a string
    • ex: let isUserLoggedIn = true;
    console.log(typeof isUserLoggedIn);
  • the == operator will perform comparisons after type coercion (it will set all variables being compared to the same type). if you don't want this to occur, use === which will not use type coercion.

  • the unary operator '+' will convert variables to a Number. 
    • ex: const str = '42';
    const strToNum = +str;
  • the unary operator '-' will do the same thing as '+' but flip the sign of the number that results.

  • the unary operator '!' will flip the boolean result of a variable.

  • the ternary operator is a simple way to write if-else statements. it's format is: condition ? expressionIfTrue : expressionIfFalse

  • put all of your conditional expressions in one bracket in an if/else statement. like if (youaredumb == True && youaresmart == False)...
  • if you want to use 'and' for an if-else, use &&
  • if you want to use 'or' for an if-else, use ||
  • you can also use ?? which will only return a result if a value is null or undefined
    • ex: const userSettings = {
                    theme: null,
                    volume: 0,
                    notifications: false,
                };
               
    let theme = userSettings.theme ?? 'light';
    console.log(theme); // light

Methods

  • to find a subsequence within a string, you can use the .indexOf("subsequence", "OPTIONAL: starting index. default is 0") method.
    • returns index where the subsequence begins. will return -1 if not in string. it is CASE-SENSITIVE

  • the .prompt() method pops up a dialog box to get an answer from the user
    • takes two arguments: prompt("message that the user will read". "what the input text field will initially have")
    • store in variable

  • the .includes() method will return true or false based on whether or not the substring in the () is in a string.
    • ex: string.includes("love island"); -> will either return true or false based on if love island is in string
    • it is case sensitive
    • you can also check if the substring is there based on a certain index by including the index as the 2nd parameter

  • the .slice() method allows you to take a portion of a string and return it as a new string without modifying the 1st string
    • takes two parameters: starting index and ending index of the portion of string you want to "slice"
    • if you omit the second parameter, it will slice everything from the starting index onwards

  • the .split() method divides a string into an array of substrings.
    • it takes one parameter, the separator. if set to "", it will separate every character in the string. if set to " " it will seperate based on where spaces are.

  • the .reverse() method reverses the elements of an array in place.

  • the .join() method creates and returns a new string by concatenating all of the elements in an array.
    • it takes one paramter, the joiner. if set to "" it will join all characters in an array into a string.

  • .toLowerCase() and .toUpperCase() will transform the entire string to either lower or uppercase

  • the .trim() method will remove any whitespace from the leading or trailing end of a string but not from the middle
    • ..trimStart() only removes whitespace from leading end.
    • .trimEnd() only removes whitespace from the trailing end.

  • the .replace() method will replace parts of a string with another string.
    • format: string.replace(searchValue, newValue); -> searchValue = part of string you want to replace with newValue
    • it will only replace the first occurrence
    • it is case sensitive

  • the .repeat() method allows you to repeat a string as many times as specified in the parameter
    • the parameter is the number of times you want it repeated

  • the Math.random() method generates a random float between 0 and less than 1
  • the formula to generate a random number between two numbers is: Math.random() * (max - min) + min;
  • the Math.min() and Math.max() will grab the smallest and largest numbers in their parameters respectively
  • to round numbers up, use Math.ceil() and to round numbers down, use Math.floor()
  • the Math.round() round the number to the nearest integer, rounding up or down as needed
  • the Math.trunc() method removes the decimal part of a number, without rounding
  • the Math.sqrt() method takes the square root of a number, and the Math.cbrt()method takes the cube root
  • the Math.abs() method will grab the absolute value of a number
  • the Math.pow() method takes two numbers in it's parameters and returns the value of the first number raised to the second

  • the .parseFloat() method allows you to take a string parameter and pull a float out of it. it starts from the beginning of the string, so if you have letters preceding a float number, it will return NaN. however, if you have letters after a float number, it will return just the float number.
  • the .parseInt() method allows you to take a string parameter and pull an integer out of it. it starts from the beginning of the string, so if you have letters preceding an integer number, it will return NaN. however, if you have letters after an integer number, it will return just the integer.
    • both of the above methods will ignore leading whitespace, and handle plus and minus' at the beginnings of numbers (i.e. they will include the minus on a minus number).

  • the .toFixed() method formats the number of numbers after the decimal point of a number (wow a lot of numbers) based on the number in it's parameter. get it? it will return a string not a number. it will also round up or down as well when cutting off numbers.
    • if you call .toFixed() with nothing in the parameters, it will return a string value of the number without any decimals and thus without any numbers after the decimal.

Numbers

  • A number data type in JavaScript can be of any type of number (float, integer, etc). They are all the same: a number
  • Multiplication and division take precedence over addition and subtraction. Add brackets if you need add/sub to be prioritized first.
  • prefix: --x. postfix: x--. basically, prefix means that the variable will immediately decrement one if you log it. but with postfix, it will only be incremented after it is called again

  • NaN stands for 'Not a Number'. Number.isNan() is a method that allows you to check for NaN values.
    • NaN is returned whenever we get undefined when we should get a number data type value (ex: running 0 / 0). it is important to account for NaN if it is a possibility in your program.

Arrays

  • arrays in javascript are the same as arrays in many other programming languages, like python. they are 0-indexed and use square brackets, [ ], for initalization and modification.

  • to add to an array, you have a few options:
    • push() can be used to add one or more elements to the END of an array. just do array.push("..."). it is similar to .append in python.
    • pop() can be used to remove the last element of the array. just do array.pop(). 1:1 the same as python.
    • unshift() can be used to add one or more elements to the START of an array. just do array.unshift("...").
    • shift() can be used to remove the first element of the array. just do array.shift().

  • array destructuring allows you to take each element out of an array and assign it to a variable, in one line.
    • let fruits = ["apple", "banana", "orange"];

let [first, second, third] = fruits;
console.log(first);  // "apple"
console.log(second); // "banana"
console.log(third);  // "orange"
  • you can skip over elements you don't want to destructure by using a comma.
    • let colors = ["red", "green", "blue", "yellow"];

let [firstColor, , thirdColor] = colors;

console.log(firstColor); // Output: "red"
console.log(thirdColor); // Output: "blue"
  • you can also set default values.
    • let numbers = [1, 2];
let [a, b, c = 3] = numbers;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
  • rest syntax is pretty neat.
    • let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let [first, second, ...rest] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(rest);   // ["orange", "mango", "kiwi"]
